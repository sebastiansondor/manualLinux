Chmod

Los permisos de archivo son las restricciones que tiene ese archivo para realizar operaciones con el. Estas operaciones pueden ser:

  • Lectura
  • Escritura(modificacion)
  • Ejecucion

Cada archivo o directorio en sistemas Linux tiene una serie de permisos que se expresan a su vez en tres tipos de usuarios distintos.

  • Usuario: el usuario al que pertenece el archivo
  • Grupo: el grupo al que pertenece el usuario
  • Otros: todos los demas usuarios

Cuando hacemos un listado de directorio, usando el comando ls -l en un sistema Linux se muestran los permisos para cada archivo o directorio.

Sintaxis

La sintaxis del comando chmod es asi:

Asignar permisos: chmod [opciones] grupoUsuario=permisos [nombreArchivo/nombreDirectorio]

Asignar permisos: chmod [opciones] grupoUsuario+permisos [nombreArchivo/nombreDirectorio]

Quitar permisos: chmod [opciones] grupoUsuario-permisos [nombreArchivo/nombreDirectorio]

Para asignar permisos a varios grupos de usuarios en una linea de comando se separan los modos por coma

Permisos:

  • C1 = {-,r} --> Puede o no leer archivos/directorios(read)

  • C2 = {-,w} --> Puede o no escribir archivos/directorios(write)

  • C3 = {-,x} --> Puede o no ejecutar los archivos/directorios(execute)

Grupos de Usuario:

  • u: usuario

  • g: grupo de usuarios del fichero

  • o: otros usuarios

  • a: todos los tipos de usuario(u,g,o)

Opciones

Como puedes ver, el comando nos ofrece muchas posibilidades. Sin embargo, las opciones mas importantes y comunes son:

  • -r: para que mire tambien en los subdirectorios de la ruta

    ubuntu@ubuntu:~$ chmod -r u=rwx archivo.txt

  • -v: para que muestre cada fichero procesado

    ubuntu@ubuntu:~$ chmod -v a+rwx archivo.txt

  • -c: es como -v, pero solo avisa de los ficheros que modifica sus permisos

    ubuntu@ubuntu:~$ chmod -c o-rwx archivo.txt

Ejemplos de uso

Asigna permisos de lectura, escritura y ejecucion para los usuarios "otros" a todos los archivos de la carpeta

ubuntu@ubuntu:~$ chmod o=rwx *
